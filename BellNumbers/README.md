## Bell Numbers

Use bottom-up DP to solve Bell numbers. This code uses Stirling numbers to calculate Bell numbers.
This is due to the following recurrence:

B(n) = sum_{k = 0}^{n} S(n,k)

__Complexity__
* DP version: polinomial.
